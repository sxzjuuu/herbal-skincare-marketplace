# AI Herbal Skincare Marketplace — Seller Upload Build

## Run
```bash
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
streamlit run app.py
```
### Features
- Marketplace with images, search, skin-type hinting
- Chat with Seller per product
- Cart → View Cart → Checkout (COD/UPI/Card mock)
- **Seller Dashboard with image upload** (adds product with uploaded image)
- Admin: Products, Orders, Users
